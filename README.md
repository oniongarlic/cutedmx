CuteDMX
=======

When all you need is a simple interface to adjust DMX lights and fixtures.

A cute and simple DMX control interface, written in C++ with Qt.

## Features

Presents a simple slider interface for adjusting DMX channel values. Think cheap DMX controller.

* Sliders for adjusting DMX channel values
* Blackout

## Supported DMX interfaces

* Only simple USB to DMX supported at this time, DOREMIDI USB-TO-DMX dongle is used for development.
